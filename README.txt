업무개선 관리 대시보드 - 서버형 (진행상황 추가 버전)

추가된 기능
- 개선기한 옆에 '진행상황' 버튼 추가
- 4월~12월 월별 업무내용 입력 가능
- 각 월별 완료 체크박스 제공
- 발의부문별 + 개선기한 순서로 자동 정렬
- 개선기한 변경 시 상세목록 순서 자동 변경
- 개선기한 필터: 2분기 / 3분기 / 4분기 / 하반기 / 제외

구성 파일
- app.py : Flask 서버
- templates/index.html : 대시보드 화면
- dashboard.db : SQLite DB
- uploads/ : 증빙자료 저장 폴더
- requirements.txt : 설치 패키지
- run_waitress.bat : 운영용 실행 배치파일
- run_flask.bat : 테스트용 실행 배치파일

실행 방법
1) 명령 프롬프트(cmd)에서 아래 순서대로 실행
cd 폴더경로
py -m pip install -r requirements.txt

2) 테스트 실행
py app.py
접속
http://127.0.0.1:5000

3) 운영용 실행
run_waitress.bat 더블클릭
또는
py -m waitress --listen=0.0.0.0:5000 app:app

저장되는 항목
- 완료여부
- 개선기한
- 진행상황(4월~12월 업무내용, 월별 체크)
- 첨부파일명 + 실제 첨부파일

주의
- 수정/삭제/첨부/진행상황 저장 시마다 관리자 비밀번호가 필요합니다.
- 첨부파일은 uploads 폴더에 저장됩니다.
- 서버 PC를 꺼두면 다른 사람은 접속할 수 없습니다.
